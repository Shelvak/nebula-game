=== Required tools ===

optipng and pngnq are provided here.

Imagemagick is also required for JPEG optimization.
swftools is also required for SWF generation.

Ubuntu: sudo apt-get install imagemagick swftools
Windows:
  ImageMagick: http://www.imagemagick.org/script/download.php
  swftools: http://www.swftools.org/download.html

All of these tools must be in your PATH: optipng, pngnq, png2swf, convert.
